## 2-2. github에 저장소 추가

내 github에 로그인해서 repository 추가한 후,

- `git init`
- `git remote add origin 해당 repositoryURL`
- `git add .`
- `git commit -m "Initialize blog"`
- `git push -u origin master`

이 단계로 하고 있었는데..
마지막 `git push` 부분에서 에러가 나오셨도다...
